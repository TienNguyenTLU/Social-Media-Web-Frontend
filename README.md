# Skyline Social Media

A modern social media platform built with Next.js, featuring Glassmorphism design and Sunset color palette.

## Features

- Glassmorphism UI with backdrop-blur effects
- Sunset-themed color system (Rose, Orange gradients)
- Fully responsive design (Mobile, Tablet, Desktop)
- Dark/Light mode toggle
- Real-time messaging interface
- Profile management
- Post creation and interaction

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Font:** Poppins (via next/font)
- **Icons:** Lucide React
- **Animations:** Framer Motion

## Getting Started

```bash
git clone https://github.com/your-username/skyline-social-media.git
cd skyline-social-media
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
